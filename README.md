## Setup

Make sure to install the dependencies with [pnpm](https://pnpm.io/installation#using-corepack):

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3212`:

```bash
pnpm dev
```

### Production

Build the application for production:

```bash
pnpm build
```
